冒泡排序的思想：将数组中的前一项和后一项依此做比较，如果后一项比前一项小，那么互换位置，以此类推，时间复杂度为n<sup>2</sup>。
![冒泡排序](3.gif)
```
var bubbleSort = (arr) => {
    for(let i = 0;i < arr.length - 1;i++) {
        for (let j = i + 1;j < arr.length;j++) {
            if (arr[i] > arr[j]) {
                let temp = arr[i]
                arr[i] = arr[j]
                arr[j] = temp
            }
        }
    }
    return arr
}
bubbleSort([4,3,1,9,6,5,2,7,8])
// [1,2,3,4,5,6,7,8,9]
```
