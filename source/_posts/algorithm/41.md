题目：实现一个函数sqrt(x)，计算并返回x的平方根，其中x为非负整数，由于返回的类型为整数，结果只保留整数部分，小数部分将被舍去。

```
let mySqrt = x => {
    var high = x, low = 1, mid
    while (low <= high) {
        mid = Math.floor((low + high)/2)
        if (mid*mid > x){
            high = mid - 1
        }
        else if (mid*mid < x){
            low = mid + 1
        }
        else {
            return mid
        }
    }
    return high;
}
mySqrt(8) //    2
```
That's all!
